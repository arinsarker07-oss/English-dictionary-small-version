# 📖 English Dictionary (Small Version)
Description:
A clean, fast, and responsive English Dictionary web application that allows users to search for word definitions, synonyms, and examples in real-time.

---

## 🚀 Live Demo
Experience the application here: [Live Preview](https://arinsarker07-oss.github.io/English-dictionary-small-version/)

---

## ✨ Key Features
- **Real-time Search:** Get definitions instantly as you type or search.
- **Phonetics & Audio:** Listen to the correct pronunciation of words.
- **Word Details:** Displays parts of speech, definitions, and usage examples.
- **Responsive UI:** Fully optimized for Mobile, Tablet, and Desktop screens.
- **Error Handling:** Graceful messages for words not found in the database.

---

## 🛠️ Technology Stack
- **HTML5:** Semantic structure.
- **CSS3:** Custom styling with a focus on modern typography.
- **JavaScript (ES6+):** Fetch API for data handling and DOM manipulation.

---

## 📦 Dependencies & Resources
This project is built using vanilla technologies to ensure high performance without external libraries.
- **API:** [Free Dictionary API](https://dictionaryapi.dev/)
- **Fonts:** Google Fonts (Inter/Roboto)
